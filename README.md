# ⌖ CLAUDE-ULTIMATE

> The ultimate Claude.ai companion extension — fully local, zero servers.

**Status:** Work in progress 🚧

## Features (planned)
- Real token counter via SSE interception
- Session + weekly usage bars with reset timers
- Warning banner at 80% and 95% usage
- Model indicator
- Mode presets (Caveman, Bullet, No-preamble, etc.)
- Response length control
- Saved custom instructions
- Prompt trimmer
- Prompt template library
- Context Passport (one-click context copy for account switching)
- One-click Markdown export
- Keyboard shortcuts
- Conversation tags

## Philosophy
Everything runs locally. Zero data collection. Every feature is opt-in.

## Install (Developer Mode — not ready yet)
1. Clone this repo
2. Go to `chrome://extensions` → Enable Developer mode
3. Load unpacked → select `extension/` folder
